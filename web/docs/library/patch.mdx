## Reference

### `from()`

<FromFunction verb="patch" />

### `update()`

```js {3}
supabase
  .from(tableName)
  .update(data)
```

It is to note that it is required to apply filters when using `.update()`. Not using filters would result in an error.

#### `data: object`

New record values.
